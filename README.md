# ORM relationship examples

### Content of project

* one-to-one (sharing one primary key)
* one-to-one (in case inheritance)
* one-to-many
* many-to-one
* many-to-many
* many-to-many (with embedded primary key)

### How to run project

* import project in IDEA
* run RelationshipsApplication class from IDEA
* or install maven and run following commands from root of project:

- `mvn install`
- `cd target`
- `java -jar relationships-0.0.1-SNAPSHOT.jar`
